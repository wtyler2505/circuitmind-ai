# Spec: Improved UI Components

## Goal
Elevate the visual fidelity of CircuitMind AI to a "High-End Cyberpunk Industrial" aesthetic using custom AI-generated assets and advanced CSS techniques.

## Current State
- **Aesthetic:** Cyberpunk/Industrial, but relies on standard SVGs and simple CSS gradients.
- **Assets:** Minimal. Placeholder text often used.
- **Feeling:** Functional but "flat".

## Target State
- **Icons:** Custom, high-fidelity, "mech" style icons (generated by Nanobanana).
- **Textures:** Seamless carbon fiber, brushed metal, and circuit board patterns for panel backgrounds.
- **Polish:** Glass refraction, neon glows, geometric corner accents, and "scanning" animations.
- **Brand:** Unique, cohesive visual identity.

## Tech Stack Changes
- **Tailwind:** Custom background patterns in `theme.extend`.
- **CSS:** Advanced `backdrop-filter` and `mask-image` usage.
- **Assets:** WebP/PNG assets stored in `public/assets/ui/`.
